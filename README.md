## ✨ Features

- [x] Photo upload with EXIF extraction
- [x] Map Visualization
- [x] Infinite scrolls
- [x] Light/Dark mode
- [x] Fully device responsive
- [x] Clean and minimalist design
- [x] i18n support (EN/ZH)
- [ ] Built-in auth
- [ ] RSS feed
- [ ] Automatic OG image generation
- [ ] AI-generated text descriptions
- [ ] and more...
